BARC OCES/DGFS 2025 Memory Based Questions

22-04-2025 17:30:00 experience pyqs

Almost negligible PYQs are available for this exam (specially for CSE), so I want to share all the memory-based questions of BARC 2025

HASHING (2Q+): Index for different keys, Chaining concepts
AVL TREES: Balance factor of a particular node after removing a node (given)
ROUTING (3Q): IP addresses contained in a given subnet, Next hop for a IP address according to routing table
PAGE REPLACEMENT ALGORITHMS (2Q+)
PROCESS SCHEDULING: RR scheduling question that asked for Average TAT & Average WT
C PROGRAMMING (9Q+): Tell o/p of given program, Choose missing line of program, Concept of pointers, arrays, malloc, struct, union & much more
FRAGMENTATION (5Q): Maximum data size which can be transferred for a given MTU, Tell data transferred with the help of values of given MF, Offset and some other fields (we multiply the offset by 8 and add the data size if MF=0)
SYNCHRONIZATION (5 to 6Q): Semaphores (maximum & minimum no. of times the threads will execute), Does the thread follow bounded waiting, mutual exclusion, progress
FORK() (2Q): No. of times print() will be executed
SORTING: How will the list look like after 2 pass of merge sort
EXPRESSION EVALUATION : Implementation of stack for a postfix expression
LOGIC GATES : Identify the gate from the given state transition diagram PROPOSITIONAL LOGIC (2Q)
SQL QUERY (3Q+)
PNC (basic) (2Q+)
STOP AND WAIT (basic formula implementation/unitary method)
TOC GRAMMAR (3Q+): Check if Left Recursive, Check if Ambiguous
DAG (Intermediate Code Generation) (2Q): Identify the number of nodes & edges for the given 3 address instructions
K-MAP: No. of 1’s in K-Map for given function
MASTER’S THEOREM (1Q)
CRC (3Q+) (a few theory-based questions regarding error detection)
CYCLOMATIC COMPLEXITY (1Q)
NUMERICAL RELATED TO LOC (1Q)
S/W ENGINEERING (theoretical) (2Q)
GRAPH THEORY (2Q+): No. of graphs possible with n vertices, Bipartite Graph LR PARSER: Comparison of no. of states in different LR parsers
CN (theoretical) (2Q+): Protocols for different layers (matching related question)
